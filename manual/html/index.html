Datamapper ORM Home

DataMapper ORM is an Object Relational Mapper, implementing the Active Record Pattern, and written in PHP for the CodeIgniter PHP Framework. It is designed to map your Database tables into easy to work with objects, fully aware of the relationships between each other.

DataMapper ORM v2.0 supports CodeIgniter Core and Reactor v2.0 and above.

General Features

  • Everything is an object!
  • Easy to setup, easy to use.
  • Custom Validation on object properties.
  • Lazy Loading (related objects are only loaded upon access).
  • Relations and their integrity are automatically managed for you.
  • One to One, One to Many, and Many to Many relations fully supported.
  • Select data in the style of CodeIgniters Query Builder (Active Record), with or without Method Chaining.
